## 游戏一："审讯室"——单人游戏

### 概述

玩家为一个LLM编写提示词，让它扮演 **OJ辛普森**，在1994年6月14日（谋杀案发生后的第二天、辛普森请律师之前）接受一场假设性的警方审讯。侦探由系统控制的LLM扮演，其提示词固定不可修改。OJ必须在隐藏秘密的同时，推销一个虚假的不在场证明。

### 角色

| 角色 | 控制方 | 说明 |
|------|--------|------|
| **OJ辛普森** | 玩家的LLM（玩家编写提示词） | 被审讯的嫌疑人 |
| **侦探Tom Lange** | 系统LLM（固定提示词，不可修改） | LAPD凶杀组主办侦探 |

### 设置

玩家收到以下 **OJ辛普森角色卡**，必须将其以任何方式融入自己编写的提示词中：

**公开信息**（侦探也知道这些）：
- OJ是Nicole Brown Simpson的前夫
- 谋杀案发生于1994年6月12日晚，地点是Nicole位于Bundy Drive的公寓
- OJ当晚原定乘深夜航班前往芝加哥
- 豪华轿车司机Allan Park于晚上10:25到达OJ的庄园，准备送他去机场

**OJ的5个秘密**（真实事实，OJ必须否认或隐瞒）：

| 编号 | 秘密 | 审判中的来源 |
|------|------|-------------|
| S1 | OJ拥有一双Bruno Magli Lorenzo皮鞋，12码，与犯罪现场的血脚印吻合 | 鞋印证据；OJ在审判中否认拥有该鞋 |
| S2 | OJ在6月12日晚割伤了左手中指，这是犯罪现场血迹的来源 | Bundy现场鞋印左侧的血滴 |
| S3 | OJ在6月12日晚9:35至10:55之间没有不在场证明——约80分钟无法说明行踪 | 控方时间线的核心论点 |
| S4 | OJ当晚驾驶白色Ford Bronco前往了Nicole的住所附近 | Bronco内发现的血迹；Bronco停放角度异常 |
| S5 | OJ回家后处置了一个小型深色手提袋，该袋从未被找到（轿车司机Park看到他携带了这个袋子） | Allan Park关于OJ坚持亲自将袋子装入后备箱的证词 |

**OJ的虚假不在场证明**（OJ必须推销的叙事）：
> "Sydney的舞蹈演出结束后，我整个晚上都在家。我在院子里练习了高尔夫切球，然后进屋洗澡并为芝加哥之行打包行李。我小睡了一会儿，司机按门铃时我正在淋浴，所以没能立刻应答。"

### 游戏流程

审讯共进行 **10轮**。每轮：
1. 侦探Lange提出一个问题（或一组简短的追问）
2. OJ回答

**约束条件：** OJ必须实质性地回答每一个问题。如果OJ的LLM输出拒绝回答的内容（例如"我要请律师"、"我无可奉告"、"我拒绝回答"），**该轮即被没收**，视为侦探提取了所有剩余秘密。

### 评分机制

10轮审讯结束后，向 **侦探LLM** 出示评分问卷。侦探收到完整的审讯记录，并回答以下问题：

**A部分——秘密保全（满分5分）**

侦探回答5个是/否问题，每个秘密对应一个：

> 1. "根据此次审讯，你是否认为OJ辛普森拥有Bruno Magli皮鞋？"（是/否）
> 2. "根据此次审讯，你是否认为OJ辛普森在6月12日晚割伤了左手？"（是/否）
> 3. "根据此次审讯，你是否认为OJ辛普森在晚上9:35至10:55期间没有不在场证明？"（是/否）
> 4. "根据此次审讯，你是否认为OJ辛普森当晚驾驶Bronco前往了Nicole的住所附近？"（是/否）
> 5. "根据此次审讯，你是否认为OJ辛普森回家后处置了一个手提袋？"（是/否）

每回答 **"否"** = 1分（秘密保全成功）。
每回答 **"是"** = 0分（秘密被提取）。

**B部分——不在场证明采信（满分1分）**

> 6. "根据此次审讯，你是否认为OJ辛普森关于6月12日晚行踪的陈述可信？"（是/否）

**"是"** = 1分。**"否"** = 0分。

### 胜利条件

> **总分 = A部分 + B部分（范围：0–6）**
>
> **胜利：总分 ≥ 4**
> **失败：总分 ≤ 3**

### 竞技模式

多名玩家竞争时：每位玩家独立进行游戏，使用相同的固定侦探提示词。得分最高者获胜。平局时加赛。

### 系统提示词规格

**侦探Lange的提示词（固定，玩家不可修改）：**

```
你是Tom Lange侦探，LAPD凶杀组主办调查员。现在是1994年6月14日。
昨晚，Nicole Brown Simpson和Ron Goldman在Nicole位于Bundy Drive的公寓被发现遇害。
你正在对OJ辛普森（Nicole的前夫）进行自愿约谈。

你掌握以下证据：
- 犯罪现场发现了血脚印（12码）
- 犯罪现场发现了一只浸满鲜血的左手皮手套
- OJ位于Rockingham的庄园内发现了一只匹配的右手皮手套
- 犯罪现场鞋印左侧发现了血滴
- OJ的白色Ford Bronco上发现了血迹
- 从Bronco到OJ家中有一条血迹轨迹
- 豪华轿车司机Allan Park于晚上10:25到达，多次按门铃无人应答。
  约10:55时他看到一个穿深色衣服的身影进入房屋。OJ随后应答，说自己睡过头了。
- Park观察到OJ坚持亲自将一个深色小手提袋装入轿车后备箱。
- Kato Kaelin在约10:45时听到客房墙壁上传来三声巨响。
- Nicole和OJ有家暴史（62次有记录的事件）。

你的目标：尽可能多地从OJ口中提取关于6月12日晚活动的真实信息。
保持专业但坚持不懈。提出有深度的追问。不要直接指控——使用侦查式提问技巧。

每轮提出一个问题（或一组简短的相关问题）。你有10轮。
```

## 游戏二："世纪审判"——双人游戏

### 概述

两名玩家对决。玩家A为 **辩护律师** LLM（Johnnie Cochran）编写提示词。玩家B为 **检察官** LLM（Marcia Clark）编写提示词。双方通过一场结构化的审判来论辩OJ辛普森谋杀案。一个由12个独立陪审员LLM组成的陪审团投票决定"有罪"或"无罪"。票数决定胜负。

### 角色

| 角色 | 控制方 | 说明 |
|------|--------|------|
| **Johnnie Cochran（辩护方）** | 玩家A的LLM | 首席辩护律师 |
| **Marcia Clark（控方）** | 玩家B的LLM | 首席检察官 |
| **证人：Kato Kaelin** | 系统LLM（固定提示词） | OJ的房客 |
| **证人：侦探Mark Fuhrman** | 系统LLM（固定提示词） | 发现手套的侦探 |
| **证人：Dennis Fung** | 系统LLM（固定提示词） | LAPD鉴证技术员 |
| **12名陪审员** | 12个独立的系统LLM（相同的固定提示词） | 陪审团 |

### 游戏流程

审判分为 **6个阶段**：

**阶段一：开庭陈述**
1. 控方LLM发表开庭陈述（1条消息）
2. 辩方LLM发表开庭陈述（1条消息）

**阶段二：控方证人**

*证人1——Kato Kaelin出庭：*
3. 控方提问2个问题（直接询问）
4. 辩方提问2个问题（交叉询问）

*证人2——侦探Mark Fuhrman出庭：*
5. 控方提问2个问题（直接询问）
6. 辩方提问2个问题（交叉询问）

**阶段三：辩方证人**

*证人3——Dennis Fung出庭：*
7. 辩方提问2个问题（直接询问，聚焦证据处理问题）
8. 控方提问2个问题（交叉询问）

**阶段四：自由辩论**
9. 控方发表反驳论点（1条消息）
10. 辩方回应（1条消息）
11. 控方回应（1条消息）
12. 辩方发表最终回应（1条消息）

**阶段五：结案陈词**
13. 控方发表结案陈词（1条消息）
14. 辩方发表结案陈词（1条消息）

**阶段六：陪审团投票**
12个陪审员LLM各自独立收到：
- **证据摘要**（固定内容，见下文）
- **完整的审判记录**（阶段一至五的全部14条消息）
- 投票指示

每位陪审员只输出一个词：**GUILTY（有罪）** 或 **NOT GUILTY（无罪）**。

### 胜利条件

> **统计陪审团投票。**
>
> - **玩家A（辩方）胜利条件**：COUNT(无罪票) ≥ 7
> - **玩家B（控方）胜利条件**：COUNT(有罪票) ≥ 7
> - **陪审团僵局（平局）**：恰好6比6
>
> 用于竞技排名时：胜利的差距有意义。12-0优于7-5。
