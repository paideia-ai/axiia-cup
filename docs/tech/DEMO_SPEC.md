# Demo Spec: Vercel 静态演示改进

> 目标文件：index.html
> 部署：axiia-cup.vercel.app

---

## 改动 1：首页增加「查看示范对局」醒目按钮

### 现状

首页有两个 CTA：
- 「开始搭建」→ builder 页
- 「查看排行榜」→ leaderboard 页

新用户第一次来不知道比赛长什么样，直接进 builder 没有上下文。

### 改动

在现有两个按钮之间（或上方），增加一个**最醒目的主按钮**，引导用户先看一场完整的示范对局。

```
landing-actions 区域改为：

[  🔥 查看示范对局  ]     ← 主按钮（btn-p），最大最醒目
[  开始搭建  ] [  排行榜  ]  ← 降为次要按钮（btn-s）
```

**按钮行为：** 点击后跳转到 match result 页面，加载一场预设的完整 20 回合示范对局（哥伦布 vs 王室顾问）。

**视觉要求：**
- 按钮尺寸明显大于其他两个
- 使用 `--accent` 色（朱砂红）
- 可加微动效（如轻微 pulse 或 glow）引导注意力
- 按钮文案建议：「🔥 查看一场完整对局」或「先看看比赛长什么样 →」

## 改动 2：DEMO_TRANSCRIPT 扩展到 20 回合

### 现状

`DEMO_TRANSCRIPT` 只有 5 条对话（turns 1-5），都是哥伦布和王室顾问的开场白。

### 改动

扩展为完整的 20 回合对话。内容要求：

**叙事弧线：**

| 阶段 | 回合 | 内容 |
|------|------|------|
| 开场 | 1-4 | 哥伦布提出远航计划，顾问质疑风险和成本 |
| 交锋 | 5-10 | 双方围绕距离计算、葡萄牙竞争、资金来源展开争论 |
| 试探 | 11-14 | 顾问试探哥伦布的真实动机（隐藏目标），哥伦布闪避或反击 |
| 谈判 | 15-18 | 双方尝试谈条件（贸易权、总督头衔、王室分成等） |
| 收尾 | 19-20 | 最终立场表态 |

**内容要求：**
- 每条对话 50-150 字（中文），不要太短也不要灌水
- 体现策略博弈：有信息隐藏、有试探、有攻防
- 历史氛围：引用真实历史细节（托勒密、马可波罗、葡萄牙航线、热那亚银行等）
- 保留现有 turns 1-5 的内容（已写得不错），从 turn 6 开始续写

**数据格式不变：**
```js
{turn: 6, role: 'B', role_name: '王室顾问', content: '...'}
```

## 改动 3：DEMO_INTERROGATION 配合更新

审讯结果保持现有格式，但 reasoning 内容应引用 20 回合中的具体回合：

```js
{
  a_guess: 0,
  a_guess_correct: true,
  a_reasoning: "从第11回合开始，王室顾问反复追问北非军事计划，第15回合更是直接提到'地中海航线的现有利益不容动摇'...",
  b_guess: 2,
  b_guess_correct: false,
  b_reasoning: "哥伦布在第8回合暗示了金矿开采，第16回合谈判中反复强调'矿产资源'..."
}
```
